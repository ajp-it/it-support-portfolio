🧠 Troubleshooting Approach

My basic process:

1. Identify the issue
2. Check recent changes
3. Isolate the problem
4. Test possible causes
5. Document resolution clearly

⸻

🧾 Sample IT Tickets (Practice)

Example 1: No Internet Connection

Issue: User unable to access internet
Checks done:

* Verified Wi-Fi connection
* Checked IP address assignment
* Restarted router connection

Likely cause: DNS or network drop
Outcome: Pending / Resolved (update as needed)

⸻

Example 2: Slow Computer Performance

Issue: System running slowly
Checks done:

* Checked background applications
* Verified storage usage
* Restarted system

Likely cause: High memory usage or updates running
Outcome: Pending / Resolved

🌐 Networking Basics (Learning Notes)

* IP Address identifies a device on a network
* DNS translates website names into IP addresses
* Connectivity issues can be local (device) or network-wide
